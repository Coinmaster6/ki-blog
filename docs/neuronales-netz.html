Wie funktioniert ein neuronales Netzwerk?

Ein neuronales Netzwerk ist ein Programm, welches die Funktionsweise eines Gehirns nachahmt. Es ist dafür verantwortlich, grosse Mengen an unstrukturierten Daten gut auszuwerten und in ihnen ein Muster zu finden. Im menschlichen Körper gibt es Nervenzellen, die durch sie sogenannten Neuronen verknüpft werden. Ein Mensch hat ziemlich viele von diesen Neuronen - Circa 20 Milliarden. All diese Neuronen sind an vielen Stellen miteinander verbunden. Diese Verbindungen nennt man Synapsen, sie bilden ein ziemlich komplexes Netzwerk. Mithilfen von diesem Netzwerk verarbeitet das menschliche Gehirn die Informationen und ermöglicht dem Menschen gewisse Dinge zu tun, zum Beispiel zu lernen oder zu lesen. Das menschliche Gehirn ist im Gegensatz zu jedem Computer theoretisch viel leistungsfähiger, nur es braucht deutlich länger, um diesen Prozess auszuführen. Die Computer kopieren dieses menschliche Netzwerk und stellen somit diese Informationsverarbeitung künstlich nach. Im Computer ist aber ein Neuron eine mathematische Formel, welche einen Input verarbeitet und daraus einen Output generiert. Ein neuronales Netz besteht meistens aus einer Inputebene, welche die Signale aufnimmt, aus mehreren Zwischenebenen, welche Verrechnungen vornehmen und aus einer Outputebene, welches das Ergebnis der Berechnung bereitstellt.

Beispiel: Ein neuronales Netz soll Äpfel von Birnen unterscheiden. Damit diese neuronalen Netze funktionieren braucht es Daten, deren Ergebnisse der Computer schon kennt, um aus ihnen zu lernen. Dieser Prozess wird Training genannt. In diesem Beispiel braucht der Computer Bilder von Äpfeln und von Birnen. Das Neuron, der Inputebene nimmt die Information auf und verarbeitet diese Information (berechnet eine Formel) und gibt das Ergebnis dann an die nächste Zwischenebene weiter. Die Ereignisse werden bis zur Outputebene weitergeleitet und entsprechen dort dann einen der beiden möglichen Outputs (Apfel oder Birne). Je nachdem, welcher Frucht die Outputneuronen mehr entsprechen, gibt der Computer Apfel oder Birne an.
